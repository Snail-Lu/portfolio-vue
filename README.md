# Vue Portfolio
基于vue2搭建的demo项目，项目中多为平时开发遇到的需求实现。[线上地址](https://snail-lu.github.io/portfolio-vue/)

## 基本使用

```bash
// 克隆代码
git clone git@github.com:Snail-Lu/portfolio-vue.git

// 安装依赖
cd portfolio-vue
npm install

// 本地运行
npm run serve

// 生产打包
npm run build

// 部署到 github pages
npm run deploy
```
